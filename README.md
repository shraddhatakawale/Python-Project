# 📊 Employee Data Analysis using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on an employee dataset using Python. The goal is to clean the data, handle missing values, transform columns, and analyze trends such as department distribution, salary patterns, experience levels, and gender representation.

## Dataset

The dataset includes employee information such as:

* Department
* Salary
* Gender
* City and State
* Join Date
* Experience
* Active Project Name
* Contact Information

## Data Processing Steps

* Loaded dataset using Pandas
* Checked dataset structure using `info()` and `columns`
* Handled missing values using mode and default values
* Converted salary column to numeric format
* Converted date columns to datetime format
* Renamed and cleaned columns
* Removed unnecessary columns

## Analysis Performed

* Department-wise employee count analysis
* Experience vs joining date analysis
* Gender distribution across departments
* Department-wise salary distribution
* Salary-based monthly bonus analysis

## Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Project Files

* `new_project.ipynb` – Python data analysis notebook
* `dataset.csv` – Dataset used for analysis
